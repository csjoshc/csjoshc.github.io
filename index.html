Learning and note taking website

What is this website used for?

This website has evolved a bit over the past few weeks, but the basic format is largely in place, aside from changes in UI (such as grid or fancier CSS styling)

  1. Sync notes between Linux and Windows since .md files are open source and cross platform. I was using MS Onenote, but it was looking to be a hassle running it in Linux, and I was just missing too many features just running it in the browser
  2. Github functions both as a remote sync service and a website host. Also, I can learn a bit of git and version control along the way
  3. Web development - by keeping my notes as a website, I am forced to learn directory and file management, as well as compiling HTML
  4. Working in IDEs: I use vscode, which is minimalistic compared to Spyder or RStudio (or Matlab) that are my background. I can learn the new IDE while taking notes in it.
  5. Easier note taking, especially for code snippets and code output. This wouldn't be true for screenshots or pictures since I would need to embed the reference and put it in the same directory, but I can manage if the number of pictures is small. I imagine I would use jupyter to take notes for Python, and Rmarkdown for R, when I have lots of graphical outputs. I am pretty familiar now with .Rmd especially from work and as I understand it I could embed non R code, such as Python, SQL and Bash, and CSS, which is already pretty sufficient.
