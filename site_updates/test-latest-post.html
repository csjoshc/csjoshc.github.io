What This Does

This page demonstrates the client-side JavaScript functionality that:

  • Automatically finds the most recent blog post
  • Creates a navigation button to the latest post
  • Works entirely on static sites like GitHub Pages
  • Uses JSON index files for efficient post discovery

How It Works

  1. Index Generation: Python script scans directories and creates JSON indexes
  2. Client-Side Discovery: JavaScript reads JSON files to find latest posts
  3. Smart Navigation: Button appears with latest post info and navigates on click
  4. Fallback Handling: Multiple fallback paths if primary navigation fails

Usage

Regenerate indexes: just update-indexes

Add to HTML: Include <script src="latest-post.js"></script>

Automatic: Button appears automatically after the main heading

Generated Files

  • year-index.json - List of all year directories
  • 8_2025/post-index.json - Posts for 2025
  • 6_2019/post-index.json - Posts for June 2019
  • latest-post.js - Client-side navigation logic
